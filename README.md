### Startup Instructions:

## To Start Server:
1. open terminal in project directory
2. cd server
3. npm install
4. npx nodemon index

## To build and start client
1. open terminal in project directory
2. cd client
3. npm install
4. npm start
5. open browser in localhost:8000
