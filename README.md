### Startup Instructions:

## To Start Server:
1. open terminal in project directory
2. cd server
3. npm install
4. npx nodemon index

## To build and start client
1. open terminal in project directory
2. cd client
3. npm install
4. npm run-script build 
5. cd build
6. npm start
7. open browser at localhost:8000
